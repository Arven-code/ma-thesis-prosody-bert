# MA Thesis – Prosody-Augmented Child Accuracy Classifier

This repository contains code for my MA thesis on multimodal modeling of parent–child interactions,
combining text and prosodic features to predict child utterance accuracy.

## Project structure
- `src/` core code
- `notebooks/` experiments and exploration
- `configs/` training configs
- `scripts/` runnable utilities
- `results/` (optional, not for large files)
